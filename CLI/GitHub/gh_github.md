# gh (GitHub) CLI

## Info
Site: https://cli.github.com

## Install
Docs: 
	- Linux: https://github.com/cli/cli/blob/trunk/docs/install_linux.md
	- Windows: https://github.com/cli/cli/blob/trunk/docs/install_windows.md
Outdated: `sudo apt install gh`
Not recommended (many issues): `sudo snap install --classic gh`
Latest: `brew install gh`

## Commands
Help: `gh --help`
Help about a command: `gh auth --help`
Version: `gh --help`

Login: `gh auth login`
Show current user: `gh auth status`

Clone a repo: `gh repo clone Aleks-Ya/postman`
