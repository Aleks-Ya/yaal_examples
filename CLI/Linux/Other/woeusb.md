# woeusb CLI

Install:
```shell
sudo add-apt-repository ppa:tomtomtom/woeusb
sudo apt-get update
sudo apt-get install woeusb
```

Create a bootable flash disk with Windows: `sudo woeusb --device Win11_25H2_EnglishInternational_x64.iso /dev/sda`
Source: https://www.techbloat.com/how-to-create-windows-11-bootable-usb-on-linux.html
