# Redis CLI

## Install
`sudo snap install redis`

## Run CLI
`redis-cli`

## Send a command as parameter
`redis-cli ping`

## Send command in CLI
1. Run CLI `redis-cli`
2. Enter command: `ping`


## Commands
Exit: `exit`
Check connection: `ping`
Save a string: `set person "John Dow"`
Read a string: `get person`
