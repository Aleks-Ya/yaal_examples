# nordvpn CLI

Connect:
```
nordvpn connect
```
List countries:
```
nordvpn countries
```
Connect to Germany:
```
nordvpn connect Germany
```
Status:
```
nordvpn status
```
Disconnect:
```
nordvpn disconnect
```
Enable VPN Kill Switch:
```
nordvpn set killswitch on
```
Show current settings:
```
nordvpn settings
```
