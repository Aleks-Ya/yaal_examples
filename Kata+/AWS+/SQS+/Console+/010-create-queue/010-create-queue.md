# 010-create-queue

## Task
Create a Queue, produce and consume message from it.

## Setup
1. Create a Queue
	1. Type: `Standard`
	2. Name: `queue-1`
2. Try
	1. Click `Send and receive messages`
	2. Send message
	3. Poll for messages
	4. Delete received messages (as sign of successul processing)

## Cleanup
1. Delete Queue `queue-1`
