# 050-asg

## Task
Run EC2 Instances as an Auto-Scaling Group.

## Steps
1. Open a new terminal
2. Change current dir
3. Set env vars
    ```shell
    set -x
    export LT_NAME=kata-lt-asg
    export ASG=kata-asg-asg
    ```
4. Create a Launch Template:
	1. Create: 
		`aws ec2 create-launch-template --launch-template-name $LT_NAME --launch-template-data file://lt-data.json`
	2. Get ID:
		```shell
		export LT_ID=$(aws ec2 describe-launch-templates --query 'LaunchTemplates[0].LaunchTemplateId' \
		  --output text --launch-template-names $LT_NAME)
		```
5. Choose the AZ:
	1. Get the default VPC ID:
		```shell
		export VPC=$(aws ec2 describe-vpcs --filters "Name=isDefault,Values=true" \
			--query "Vpcs[0].VpcId" --output text)
		```
	2. Get subnets:
		```shell
		export SUBNETS=$(aws ec2 describe-subnets --filters "Name=vpc-id,Values=$VPC" \
		  --query "Subnets[*].SubnetId" --output text | tr '\t' ',')
		```
6. Create an ASG:
	1. Create:
		```shell
		aws autoscaling create-auto-scaling-group \
			--auto-scaling-group-name $ASG \
			--launch-template LaunchTemplateName=$LT_NAME,Version=1 \
			--min-size 1 --max-size 3 --desired-capacity 1 \
			--vpc-zone-identifier "$SUBNETS"
		```
	2. List activities: `aws autoscaling describe-scaling-activities --auto-scaling-group-name $ASG`
6. Run an EC2 Instance:
	1. Run: `aws ec2 run-instances --launch-template LaunchTemplateId=$LT_ID,Version='$Default'`
	2. Get Instance IDs: 
		```shell
		export I_IDS=$(aws ec2 describe-instances --output text \
  		  --filters "Name=tag:aws:ec2launchtemplate:id,Values=$LT_ID" \
  		  --query "Reservations[].Instances[].InstanceId")
		```

## Cleanup
1. Delete the ASG: `aws autoscaling delete-auto-scaling-group --auto-scaling-group-name $ASG --force-delete`
2. Delete the Launch Template: `aws ec2 delete-launch-template --launch-template-name $LT_NAME`
3. Close the terminal
